How does this work?

  • Map

    Pick a metric to color countries. Use the lasso to select countries; only selected ones show in the plots below.

  • Scatter plot

    X = map attribute, Y = scatter attribute.

    Hover or select on the map or parallel plot to highlight the same country here.

  • Parallel coordinates plot

    Each line is a country. Drag on an axis to use the brush filtering tool. Filtering here also filters the other plots.

Hovering a country in the map or the scatter plot highlights it in all three.
