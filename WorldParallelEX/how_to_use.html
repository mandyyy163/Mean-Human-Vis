How to use this dashboard

Basic navigation

  • Use the dropdowns in the sidebar to choose the map (X), scatter (X, Y), and bubble attributes (X, Y, Z). Each chosen attribute carries over to the other plots.
  • The same items are linked across the map, scatter plot, bubble chart, and parallel coordinates (highlighted when hovered).

Sidebar buttons

  • Map: Select attribute for the map (one-dimensional). This is the X-axis attribute for the scatter plot and bubble chart.
  • Scatter: Select attribute for the scatter plot's Y-axis. This is also the Y-axis attribute for the bubble chart.
  • Bubble: Select attribute for the bubble chart's Z-axis. This is the size attribute for the bubble chart.
  • Reset all: Reset all filters and selection.

Functions

  • Selection: Use the lasso or box tool to select a group of countries. Other countries are greyed out. The parallel coordinates plot has a brush tool to filter the data, try it by clicking anddragging on an axis.
  • Highlighting: When hovering a country in the map, scatter plot bubble chart, it highlights in all plots.
  • Other plotly tools: Tools in the top right corner of the plots can be used to zoom, pan, reset the view, etc.
